# 🛒 CartShop - MERN Stack E-Commerce App
CartShop is a full-stack **e-commerce website** built with the MERN stack (MongoDB, Express.js, React.js, Node.js).  
It works similar to Flipkart/Amazon where users can browse categories, view product details, add items to their cart or wishlist, and place orders.  

The project also includes an **admin panel**, where admins can manage product categories and product details (add, edit, delete, upload images).

---

## 🚀 Features

### 👤 User Features
- 🔍 Browse products by category
- 📄 View product details page
- 🛒 Add to cart functionality
- ❤️ Wishlist support
- 🔑 User signup/login with validation  
  (minimum 8 characters, includes letters, numbers, and symbols)
- 👤 Profile/account page
- 📱 Responsive design

### 🛠 Admin Features
- 🔐 Admin login authentication
- ➕ Add, edit, delete product categories
- 📦 Add products with:
  - Name
  - Title
  - Brand
  - Price
  - Rating/Message
  - Image upload (Multer)
- 📋 Manage products (edit/delete)
- Protected **Admin Dashboard**

### 🧰 Work in Progress
- [ ] Secure JWT-based authentication  
- [ ] Payment gateway integration (Razorpay/Stripe)  
- [ ] Order management panel  
- [ ] User order history & invoices  
- [ ] Email notifications (order confirmation, etc.)    
- [ ] Enhanced UI/UX with animations  

---

## 🧑‍💻 Tech Stack

- **Frontend:** React.js, Bootstrap 5, React Router, Axios  
- **Backend:** Node.js, Express.js  
- **Database:** MongoDB (Mongoose ODM)  
- **Image Upload:** Multer  
- **Authentication:** LocalStorage (JWT planned)

## ⚙️ Installation & Setup

### Prerequisites
- Node.js installed  
- MongoDB running locally on `mongodb://localhost:27017/`  
- Vite installed globally (optional)  

### Clone Repository
```bash
git clone https://github.com/your-username/cartshop.git
cd cartshop

Backend Setup
cd Backend
npm install
node server.js

Frontend Setup
cd client
npm install
npm run dev
